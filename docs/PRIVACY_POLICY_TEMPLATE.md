# Шаблон политики конфиденциальности (черновик)

**Не является юридической консультацией.** Согласуйте текст с юристом под вашу юрисдикцию и роль (разработчик приложения / оператор сервера).

Опубликуйте на стабильном **HTTPS** URL и укажите его в Google Play Console и при необходимости в приложении.

---

## 1. Введение

Кратко: название приложения, для чего оно служит (клиент для серверов Rocket.Chat), кто оператор политики (ваша организация / контакты).

## 2. Какие данные обрабатываются

- **Данные сервера**: URL и учётные данные, которые пользователь вводит для подключения к своему или корпоративному серверу Rocket.Chat.
- **Сообщения и файлы**: обрабатываются сервером Rocket.Chat в соответствии с настройками этого сервера; приложение отображает и отправляет данные по запросу пользователя.
- **Push-уведомления**: при использовании Firebase Cloud Messaging — токен устройства может передаваться на сервер Rocket.Chat для доставки уведомлений; метаданные обрабатываются по правилам Google и вашего сервера.
- **Звонки**: при использовании встроенных звонков — доступ к камере/микрофону по разрешению пользователя в рамках сеанса связи.

## 3. Передача третьим лицам

Укажите: Google (FCM, при использовании), хостинг сервера Rocket.Chat пользователя, при необходимости — провайдеры аналитики (если подключите позже).

## 4. Хранение и безопасность

Локальное хранение на устройстве (кэш, база, токены сессии) — в общих чертах. Передача по сети: предпочтительно HTTPS; если допускается HTTP к определённым серверам — опишите это явно.

## 5. Права пользователя

Обращения, удаление аккаунта на стороне сервера, отзыв согласия — по применимому закону (GDPR и т.д.).

## 6. Изменения политики

Как уведомляете об обновлениях и дата последней редакции.

---

Замените плейсхолдеры, удалите неприменимые разделы и добавьте контакты: `privacy@example.com` → ваш адрес.
